HTML & CSS Table Reference

HTML Tags

Tag Name Description
<!DOCTYPE HTML> Document Type Document type being used. Starts the HTML file.
<p> Paragraph Used for paragraphs
<strong> Strong Imply importance, urgency or seriosness
<em> Emphesis Emphesis a word or sentence
<address> Address Defines the contact information for the author/owner of a document or an article.
<blockquote> Block Quote Used to define a block of text which is quoted from another source.
<q> Quote Indicates that the enclosed text is a short inline quotation
<var> Variable Used to defines a variable in programming or in a mathematical expression.
<br> Single Line Break Produces a line break in text
<a> Anchor Tag Anchor tag for a link
<base> Base To specify a base URI, or URL, for relative links. This URL will be the base URL for every link on the page and will be prefixed before each of them. For example, if the URL specified by the base tag is “www.xyz.com” then every other URL on the page will be prefixed by, “www.xyz.com/”
<ul> Unordered List Creates an unordered list (bulletpoints)
<ol> Ordered List Creates an ordered list (numbered)
<li> List Creates a list
<dl> Description List Represents a description list
<dt> Description Term It is used in a description list to define a single term, which will be described with a description detail.
<div> Division Block-level element that only represents its child elements
<section> Section Defines a section in a document
