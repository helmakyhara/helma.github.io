Text Styles

This section lists out the text style to my preference.

H1: Main Heading

  • Font-weight = 700 (bold)
  • Font-size = 26px
  • Font-family = Cardo

H2: Sub Heading

  • Font-weight = 500
  • Font-size = 18px
  • Font-family = Cardo

P: Paragraph Text

  • Font-weight = 400 (regular)
  • Font-size = 14px
  • Font-family = Mooli
